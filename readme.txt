Esse serviço fica polando um bot do telegram esperando pelo comando /buscar. Ao receber,


Variáveis que precisam ser setadas:
- TOKEN: token para comunicação com bot do telegram;
- URLRESTWIKI: url de busca REST;
- Opcional: usuario e senha do gmail para envio de emails.

Exemplo no Windows:
set URLRESTWIKI=
set USUARIOMAIL=
set SENHAMAIL=
set TOKEN=
